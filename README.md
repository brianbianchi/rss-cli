# rss-cli

```console
$ readr --help

    [NO ARGUMENT]
		Read feeds.

    --src, -s
		Print the RSS URLs for your feed.

    --add [URL], -a [URL]
		Add a RSS URL for your feed (https://www.reddit.com/.rss).

    --remove [URL], -r [URL]
		Remove a RSS URL for your feed (https://www.reddit.com/.rss).

    --path, -p
		 Print the path of the text file containing your feed URLs.

```
